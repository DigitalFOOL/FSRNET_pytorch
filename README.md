# FSRNet Pytorch
This is my personal implement of FSRNET(CVPR 2018).

Based on [WaveletSRNet](https://github.com/hhb072/WaveletSRNet/), I altered the code by adopting FSRNet network structure.

* Python 3.6
* Pytorch 0.4.1 GPU required
